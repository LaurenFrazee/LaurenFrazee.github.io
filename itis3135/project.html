Project Overview

The portfolio web page serves as a platform for the Product Manager to share his knowledge and insights through webinars and blogs. It also provides an avenue for viewers and readers to provide their feedback, fostering an interactive learning environment. The portfolio web page for the Microsoft Product Manager is designed to be a comprehensive platform for sharing knowledge through webinars and blogs, while also encouraging viewer interaction through a feedback system. The addition of a gallery, an events calendar, and a resources section further enhances the user experience by providing visual insights into the Product Manager’s work, keeping users informed about upcoming events, and offering valuable resources.

Page Design

  • Webinars: This section will highlight the webinars conducted by the Product Manager. It will include details such as the webinar topic, date, duration, and a brief description. Users will have the option to watch these webinars directly from the site
  • Blogs: This section will feature blogs written by the Product Manager. Each blog post will include the title, publication date, and a short summary. Users can read the full blog post by clicking on the provided link.
  • Gallery: This section will display a collection of images related to the webinars and blogs, or any other relevant content. The gallery will provide a visual representation of the Product Manager’s work and achievements.
  • Events Calendar: A calendar that displays upcoming webinars or events that the Product Manager will be participating in. This feature allows users to stay informed about future events and plan their participation accordingly.
  • Feedback: A crucial component of the site is the feedback mechanism. After viewing a webinar or reading a blog, users can leave their comments and suggestions. This feedback will help the Product Manager improve future webinars and blogs.This page will include input validation using dynamic functionality.

Dynamic Functionalilty

The form pages will incorporate input validation using jQuery code.This ensures that the data entered by the user meets certain criteria before it is submitted to the server. In the gallery section, when a user hovers over an item, an overlay will appear. This overlay will contain information about the author and the title of the blog post associated with that item. This feature enhances user interaction and provides additional information without requiring a click-through. The inspiration for this feature comes from www.greatbigcanvas.com
